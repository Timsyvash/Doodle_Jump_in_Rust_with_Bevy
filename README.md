# Doodle_Jump_in_Rust_with_Bevy
Я зробив гру Doodle Jump на Rust з Bevy

# Про Doodle Jump
Doodle Jump — це відеогра- платформер 2009 року , розроблена та видана Ігорем та Марко Пушеняк, які складають Lima Sky LLC. Гра була випущена для Windows Phone , iOS , BlackBerry , Android , Java Mobile , Nokia Symbian та Xbox 360 для Kinect. Вона була випущена по всьому світу для iOS у березні 2009 року, для Android та Blackberry — 2 березня 2010 року, для Symbian — 1 травня 2010 року, для Windows Phone 7 — 1 червня 2011 року, для iPad — 1 вересня 2011 року та для Windows Phone 8 — 21 серпня 2013 року. З моменту свого випуску гра була загалом добре сприйнята. Doodle Jump був відомий своїми темпами продажів за стандартами App Store , які протягом чотирьох місяців поспіль щодня продавалися в 25 000 копій, а пізніше їх обігнала Angry Birds . Станом на грудень 2011 року було продано 10 мільйонів копій гри через iTunes та Google Play.
# Про Rust
Rust — це сучасна компільована мова програмування загального призначення, орієнтована на безпеку, швидкість і паралелізм. Вона поєднує високу продуктивність (подібну до C/C++) із гарантованою безпекою пам'яті без використання збирача сміття, завдяки унікальній системі «власності» (ownership) та «позичання» (borrowing). 
Де використовують Rust:
-Системне програмування: Розробка операційних систем, драйверів, ядра.
-Веб-сервіси: Високонавантажені сервіси та WebAssembly.
-Вбудовані системи (Embedded): Розробка для мікроконтролерів.
-Ігрова індустрія: Створення ігрових рушіїв.
# Про Bevy
Bevy — це сучасний, високопродуктивний ігровий рушій (engine) з відкритим вихідним кодом, написаний мовою програмування Rust Reddit. Він використовує архітектуру Entity Component System (ECS) для розробки 2D та 3D ігор, вирізняючись простотою використання, швидкістю та модульністю, що робить його перспективним вибором для розробників.
Основні характеристики Bevy:
Мова: Написаний та використовується з мовою Rust, що забезпечує безпеку пам'яті без збирача сміття.
Архітектура: Повністю базований на ECS (Entity-Component-System), що забезпечує ефективне керування даними та високу продуктивність.
Спрямованість: Підходить як для 2D, так і для 3D проектів.
Open Source: Безкоштовний, з відкритим кодом, що активно розвивається спільнотою. 
Bevy є чудовим вибором для тих, хто хоче вивчати розробку ігор на Rust.
